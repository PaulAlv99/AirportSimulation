# OpenFlights Route Template Data

This directory contains static OpenFlights data used as route, airline, and aircraft templates for the simulation:

- `routes.dat`
- `airlines.dat`
- `planes.dat`

Source: https://openflights.org/data.php

The OpenFlights route data is historical route-template data, not live schedule data. The simulation generates flights, passengers, baggage, gates, and ground operations from these templates.

OpenFlights documents these databases under the Open Database License and Database Contents License. Keep attribution and license notes when redistributing derived data.
